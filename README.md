# Aura Retail OS  
Designing an Autonomous Modular Smart-City Retail Infrastructure

---

## Project Overview

Aura Retail OS is a modular and adaptive retail kiosk system designed for smart city environments such as hospitals, metro stations, universities, and emergency zones.

The system simulates intelligent kiosks capable of:
- Handling dynamic pricing
- Managing inventory efficiently
- Responding to emergencies
- Recovering from failures automatically

---

## Key Features

- Inventory Management System  
- Purchase and Restock Functionality  
- Emergency Mode with Dynamic Pricing  
- Event Notification System (Low Stock Alerts)  
- Failure Handling and Recovery  
- Modular and Scalable Architecture  

---

## Concepts Implemented

### Object-Oriented Principles
- Encapsulation  
- Abstraction  
- Inheritance  
- Low Coupling  

### Design Patterns Used
- Strategy Pattern (Dynamic Pricing)  
- Command Pattern (Transactions)  
- State Pattern (Kiosk Modes)  
- Observer Pattern (Event System)  
- Chain of Responsibility (Failure Handling)  
- Memento Pattern (Rollback System)
